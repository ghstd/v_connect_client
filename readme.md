## How to run

1. [Install ruby](https://rubyinstaller.org/downloads/)
   - [installation instructions](https://youtu.be/6_ek4hokiak?si=GPFuqgVqyPVX_aR4&t=209)
2. Install gems:
   - gem install fxruby
   - gem install websocket-client-simple
   - gem install ffi
3. Create .bat file:

```
@echo off
ruby path\to\your\app.rb
pause
```
